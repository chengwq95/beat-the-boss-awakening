# 👊 暴打老板：打工人觉醒

一款零依赖、可离线游玩的中文点击放置游戏。疯狂点击老板、积攒怒气、命中弱点、使用技能和应急道具，并通过辞职周目不断觉醒变强。

## 在线游玩

[在线游玩：beat-the-boss-awakening.pages.dev](https://beat-the-boss-awakening.pages.dev/)

## 本地运行

直接用浏览器打开 `index.html` 即可，无需安装依赖或启动服务。

## 玩法

- 点击老板造成伤害并获得奖金；购买职场军火库装备提升点击伤害、自动伤害、暴击率和额外收益。
- 积攒怒气后释放技能：连续欧拉、加班爆发、劳动仲裁，分别强化点击、自动伤害或直接扣除老板生命。
- 战斗中会随机出现弱点，命中后造成高额伤害；老板也会读条释放技能，需要及时打断。
- 每 5 关为狂暴关，需要在限时内击败老板；突破后可获得永久增强奖励。
- 应急道具可临时提升点击、自动伤害或冻结时间，帮助处理卡关和狂暴关。
- 达到条件后可以开启“辞职周目”，重置关卡与商店，换取永久全局加成。
- 进度保存在浏览器的 `localStorage`，支持自动存档和离线收益。

## 部署

这是静态站点。Cloudflare Pages 的构建设置应为：

- Framework preset：`None`
- Build command：留空
- Build output directory：`/`

如需把项目放在仓库子目录中，请将根目录设置为该子目录，或把 `index.html` 移到仓库根目录。

## 技术

原生 HTML、CSS、JavaScript；样式、逻辑和音效都内置在 `index.html` 中。音效由 Web Audio API 实时生成，不依赖任何第三方资源。
